# Rock-paper-scissor-game
Rock-paper-scissor game for CGV Course work 2025
How To Run ?
01.Clone or download the project.

02.Install the requirements:

		pip install -r requirements.txt

03.Run The Game :- 
     Open Your Command Prompt Inside the  folder of the python_RPS.py file located on,
      Then run this command,
		 python python_RPS.py


How to Play

>>>Make a hand gesture for Rock, Paper, or Scissors.

>>>Say "Rock, Paper, Scissors, Shoot!" and press s to capture your gesture.

>>>The computer randomly selects a move.

>>>The program determines the winner and displays the result.

>>>Press q to quit the game.
